# 计时系统发布检查清单

## 1. 代码与测试

- [ ] `npm test` 通过
- [ ] `npm run test:replay` 通过
- [ ] `npm run build` 通过
- [ ] `cargo test` 通过

## 2. 数据口径抽样

- [ ] 应用排行中无系统进程（含 `PickerHost`）
- [ ] 同应用变体未拆分为多条（含 `douyin_tray` / `douyin_widget`）
- [ ] 应用分布总和与总计时长一致
- [ ] 最小记录阈值仅影响时间轴，不影响统计总时长

## 3. UI 抽样

- [ ] 概览页不显示内部诊断模块
- [ ] 历史页不显示内部诊断模块
- [ ] 活跃应用状态与当前前台应用一致（非 AFK 时）

## 4. 发布前确认

- [ ] 文档勾选状态已更新（治理方案 + 数据契约）
- [ ] 风险变更点已记录（规则变更、过滤变更、归并变更）
- [ ] 回滚清单已准备
